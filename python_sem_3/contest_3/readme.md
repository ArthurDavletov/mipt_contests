### B. Генератор 220v

> Статус: Полное решение. Видимые вердикты: OK, PE. Посылка: есть.

Что будет если смешать:

ЭТА ЗАДАЧА!

Напомним, что email адрес представляется как local@domain.

Вам нужно написать генератор, который умеет генерировать email адрес. Но не все так просто. Генератор должен быть умный и уметь подстраиваться под наши хотели, чтобы, например, уметь генерировать адреса с разными айдишниками по маске 'chupov[1-9]*@mail\.ru':

Генератор может принимать на вход четыре опциональных аргумента: regex, domain, local, size.

regex - строка с регулярным выражением для всего адреса

domain - строка с регулярным выражением для имени домена

local - строка с регулярным выражением для локального имени

size - количество элементов (если не указан, то генератор должен быть бесконечным)

Обратите внимание, ни один из параметров НЕ ЯВЛЯЕТСЯ обязательным. То есть может быть просто вызов for i in email_generator()

#### B. Примечания
