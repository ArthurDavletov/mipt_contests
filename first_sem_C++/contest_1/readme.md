# Контест 1

Ссылка на контест: [Яндекс.Контест](https://contest.yandex.ru/contest/52447/problems/A/)

## Задания

### A. Hello, World

Ваша первая задача в две тысячи двадцать шестом году.

В ваших силах сделать все, чтобы она не стала последней.

Итак, выведите "Hello, World!".

#### A. Формат ввода

Вводить ничего не требуется.

#### A. Формат вывода

Выведите "Hello, World!".

#### A. Пример

- **Вывод**

```text
Hello, World!
```

### B. Гипотенуза

Дано два числа $a$ и $b$. Найдите гипотенузу треугольника с заданными катетами.

#### B. Формат ввода

В двух строках вводятся два числа (числа натуральные, не превышающие 1000).

#### B. Формат вывода

Выведите ответ на задачу.

#### B. Пример

- **Ввод**

```text
3
4
```

- **Вывод**

```text
5
```
