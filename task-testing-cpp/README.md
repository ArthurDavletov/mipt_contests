# task-testing-cpp

## Установка и запуск

Вы можете с лёгкостью установить данный проект и протестировать у себя, запустив следующие команды:

```sh
cd test_repo
git clone git@gitlab.akhcheck.ru:artur.davletov/first_repo.git
git branch task-testing-cpp origin/task-testing-cpp
git checkout task-testing-cpp
cd task-testing-cpp
./build.sh
```
