## Задания

### A. Побочная диагональ с использованием массивов

Напишите функцию, которая по введенному числу $n$ выводит числа таблицей по следующему правилу:

- числа на диагонали, идущей из правого верхнего в левый нижний угол, равны 1;

- числа, стоящие выше этой диагонали, равны 0;

- числа, стоящие ниже этой диагонали, равны 2.

#### A. Формат ввода

Программа получает на вход число $n$ ($n < 100$).

#### A. Формат вывода

Необходимо вывести таблицу. Числа разделяйте одним пробелом.

#### A. Пример

- **Ввод**

```text
2
```

- **Вывод**

```text
0 1 
1 2
```

### B. Состязание

В метании молота состязается $n$ спортcменов. Каждый из них сделал $m$ бросков. Победитель определяется по лучшему результату. Определите количество участников, а так же самих участников состязаний, которые разделили первое место, то есть определите количество строк в массиве, которые содержат значение, равное наибольшему.

#### B. Формат ввода

Программа получает на вход два числа $n$ и $m$, являющиеся числом строк и столбцов в массиве. Далее во входном потоке идет $n$ строк по $m$ чисел, являющихся элементами массива.

#### B. Формат вывода

Сначала программа выводит количество спортсменов, показавших наилучший результат, затем – их номера в порядке возрастания. Не забудьте, что строки (спортсмены) нумеруются с 0.

#### B. Пример

- **Ввод**

```text
5 4
1 1 1 0
0 1 2 2
2 0 1 0
0 2 0 2
0 0 0 0
```

- **Вывод**

```text
3
1 2 3
```

### C. Треугольник Паскаля

Напишите функцию, которая по двум числам $n$ и $m$ создает двумерный массив [$n$][$m$] и заполняет его по следующим правилам: числа, стоящие в строке 0 или в столбце 0 равны 1 $(A[0][j]=1, A[i][0]=1)$. Для всех остальных элементов массива $A[i][j]=A[i-1][j]+A[i][j-1]$, то есть каждый элемент равен сумме двух элементов, стоящих слева и сверху от него.

#### C. Формат ввода

Программа получает на вход два числа $n < 100$ и $m < 100$.

#### C. Формат вывода

Выведите данный массив.

#### C. Пример

- **Ввод**

```text
3 3
```

- **Вывод**

```text
1 1 1 
1 2 3 
1 3 6
```

### D. Заполнение диагоналями с использованием массивов

#### D. Формат ввода

Программа получает на вход два числа $n$ и $m$ ($1 \leq n, m \leq 100$).

#### D. Формат вывода

Программа должна вывести числа таблицей. Числа разделяйте одним пробелом.

#### D. Пример

### E. Сапер

Напишите программу, отображающую игровое поле для игры "Сапер".

#### E. Формат ввода

Даны числа $N$ и $M$ (целые, положительные, не превышают 32) – количество строк и столбцов в поле соответственно, далее число $W$ (целое, неотрицательное, не больше 1000) – количество мин на поле, далее следует W пар чисел, координаты мины на поле (первое число – строка, второе число – столбец).

#### E. Формат вывода

Требуется вывести на экран поле. Формат вывода указан в примере.

#### E. Пример 1

- **Ввод**

```text
3 2
2
1 1
2 2
```

- **Вывод**

```text
* 2 
2 * 
1 1
```

#### E. Пример 2

- **Ввод**

```text
2 2
0
```

- **Вывод**

```text
0 0 
0 0
```

### F. Спираль

Напишите функцию, которая заполняет массив размера $N \times N$ единичками по спирали (начиная с верхнего левого угла по часовой стрелке, см. пример).

#### F. Формат ввода

С клавиатуры вводится число $N$ (нечетное, натуральное и не превышающее 50).

#### F. Формат вывода

Требуется вывести на экран построенную спираль.

#### F. Пример

- **Ввод**

```text
7
```

- **Вывод**

```text
1111111
0000001
1111101
1000101
1011101
1000001
1111111
```

### G. Бинарный поиск

Реализуйте алгоритм бинарного поиска.

#### G. Формат ввода

В первой строке входных данных содержатся натуральные числа $N$ и $K$ $(0<N, K\leq 10^6)$. Во второй строке задаются $N$ элементов первого массива, отсортированного по возрастанию, а в третьей строке — $K$ элементов второго массива. Элементы обоих массивов - целые числа, каждое из которых по модулю не превосходит $10^9$.

#### G. Формат вывода

Требуется для каждого из $K$ чисел вывести в отдельную строку «YES», если это число встречается в первом массиве, и «NO» в противном случае.

#### G. Пример 1

- **Ввод**

```text
10 4
-7 -3 0 6 7 10 15 17 18 21
16 -7 30 10
```

- **Вывод**

```text
NO
YES
NO
YES
```

#### G. Пример 2

- **Ввод**

```text
12 7
1 3 5 6 8 10 20 28 29 31 33 40
3 23 50 20 45 40 26
```

- **Вывод**

```text
YES
NO
NO
YES
NO
YES
NO
```

### H. НОД и НОК

Вычислите НОД и НОК двух чисел.

#### H. Формат ввода

На вход подается два числа $x, y$ через пробел. $0 < x, y < 2^{31} - 1$

#### H. Формат вывода

Выведите два числа, разделенных через пробел: НОД($x, y$) и НОК($x, y$).

#### H. Пример

- **Ввод**

```text
7 3
```

- **Вывод**

```text
1 21
```
