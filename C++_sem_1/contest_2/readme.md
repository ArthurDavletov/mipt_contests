## Задания

### A. Максимум из трех чисел

Даны три целых числа, каждое записано в отдельной строке. Выведите наибольшее из данных чисел.

#### A. Формат ввода

На вход подаются 3 числа не большие 10000.

#### A. Формат вывода

Единственное число – искомый ответ.

#### A. Пример

- **Ввод**

```text
1
2
3
```

- **Вывод**

```text
3
```

#### A. Примечания

Friendly reminder: при решении можно использовать только арифметические операции (+, -, *, /, %) и и условные конструкции.

### B. Шоколадка

Требуется определить, можно ли от шоколадки размером n × m долек отломить k долек, если разрешается сделать один разлом по прямой между дольками (то есть разломить шоколадку на два прямоугольника).

#### B. Формат ввода

Вводятся 3 числа: n, m и k, причем k не равно n × m. Гарантируется, что количество долек в шоколадке не превосходит 40000.

#### B. Формат вывода

Программа должна вывести слово YES, если возможно отломить указанное число долек, в противном случае вывести слово NO.

#### B. Пример

- **Ввод**

```text
4
2
6
```

- **Вывод**

```text
YES
```

#### B. Примечания

Friendly reminder: при решении можно использовать только арифметические операции (+, -, *, /, %) и и условные конструкции.

### C. Сложное уравнение

Решить в целых числах уравнение (ax + b): (cx + d) = 0.

#### C. Формат ввода

Вводятся 4 числа: a, b, c и d; c и d не равны нулю одновременно.

#### C. Формат вывода

Необходимо вывести все целочисленные решения, если их число конечно, "NO" (без кавычек), если целочисленных решений нет, и "INF" (без кавычек), если их бесконечно много.

#### C. Пример

- **Ввод**

```text
1
1
2
2
```

- **Вывод**

```text
NO
```

#### C. Примечания

Friendly reminder: при решении можно использовать только арифметические операции (+, -, *, /, %) и и условные конструкции.

### G. Второй максимум - 2

Последовательность состоит из натуральных чисел и завершается числом 0. Определите значение второго по величине элемента в этой последовательности, то есть элемента, который будет наибольшим, если из последовательности удалить наибольший элемент.

Числа, следующие за числом 0, считывать не нужно.

#### G. Формат ввода

Вводится последовательность целых чисел от 1 до 10^9, оканчивающаяся числом 0 (само число 0 в последовательность не входит).

#### G. Формат вывода

Выведите ответ на задачу.

#### G. Пример

- **Ввод**

```text
1
7
9
0
```

- **Вывод**

```text
7
```

### I. N-е число Фибоначчи

Последовательностью Фибоначчи называется последовательность чисел a_0, a_1,..., a_n,..., где a_0 = 0, a_1 = 1, a_k = a_k-1 + a_k-2 (k > 1).

Найдите N-ое число Фибоначчи.

#### I. Формат ввода

На вход программы поступает целое число 0 <= N <= 30.

#### I. Формат вывода

Требуется вывести N-е число Фибоначчи.

#### I. Пример

- **Ввод**

```text
3
```

- **Вывод**

```text
2
```

### J. Разрезание на квадраты

Полоска бумаги имеет размеры A × B. Каждый раз от нее отрезается квадрат максимального размера до тех пор, пока не получится квадрат. Сколько квадратов получится?

#### J. Формат ввода

Программе даны числа 1 <= A, B <= 10^9.

#### J. Формат вывода

Требуется вывести количество квадратов.

#### J. Пример

- **Ввод**

```text
15 3
```

- **Вывод**

```text
5
```

### K. Сокращение дроби (TL)

Дана дробь \fracab. Требуется ее сократить, то есть записать это же число в виде \fraccd, где c — целое число, d — натуральное число и d минимальное возможное.

#### K. Формат ввода

Вводятся два целых числа -10^9 <= a <= 10^9 и -10^9 <= b <= 10^9.

#### K. Формат вывода

Выведите два числа c и d через пробел.

#### K. Пример

- **Ввод**

```text
3 6
```

- **Вывод**

```text
1 2
```

### L. Побочная диагональ

Дано число n, оно задает размер таблицы n × n. Выведите числа таблицей по следующему правилу:

- числа на диагонали, идущей из правого верхнего в левый нижний угол, равны 1;

- числа, стоящие выше этой диагонали, равны 0;

- числа, стоящие ниже этой диагонали, равны 2.

#### L. Формат ввода

Программа получает на вход число n (n < 100).

#### L. Формат вывода

Необходимо вывести таблицу. Числа разделяйте одним пробелом.
